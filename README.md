# VibeCoding Website

Eine moderne Landing Page gebaut mit Next.js, TypeScript und Tailwind CSS — deployed auf Vercel.

## Features

- Responsive Design mit dunklem Theme
- Animierte Hero-Section mit Gradient-Effekten
- Features, Projekte und Kontakt-Sektionen
- Optimiert für Vercel Edge Deployment

## Tech Stack

- [Next.js 16](https://nextjs.org/)
- [React 19](https://react.dev/)
- [TypeScript](https://www.typescriptlang.org/)
- [Tailwind CSS 4](https://tailwindcss.com/)
- [Vercel](https://vercel.com/)

## Lokale Entwicklung

```bash
npm install
npm run dev
```

Öffne [http://localhost:3000](http://localhost:3000) im Browser.

## Deployment

Das Projekt ist für Vercel optimiert. Bei jedem Push auf den `main` Branch wird automatisch deployed.

```bash
vercel --prod
```
